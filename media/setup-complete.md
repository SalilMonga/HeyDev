# You're all set!

HeyDev is now configured and watching your terminals.

### Next steps:

1. **Open a terminal** and start a Claude Code or Codex session
2. **Watch the tab title** update with status indicators
3. **Wait for a notification** when your AI agent needs input (default: 60s delay)

### Customize in Settings:

| Setting | What it does |
|---------|-------------|
| Notification delay | How long to wait before alerting (0 = immediate) |
| Emojis | Change the working/waiting indicators |
| Show notifications | Toggle notifications on/off |

### Tips:

- Each session gets a unique 4-character tag so you can tell them apart
- Quick Reply lets you send "yes", "continue", etc. without switching terminals
- Notifications auto-cancel if you switch to the terminal or the AI starts working
