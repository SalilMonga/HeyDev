# Install jq

HeyDev uses `jq` to parse JSON from your AI CLI sessions.

### macOS (Homebrew)
```bash
brew install jq
```

### Ubuntu / Debian
```bash
sudo apt-get install jq
```

### Windows
```bash
choco install jq
```

### Verify installation
```bash
jq --version
```

Once installed, proceed to the next step.
