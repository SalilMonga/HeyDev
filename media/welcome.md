# Welcome to HeyDev

**Stop tab-switching. Start building.**

HeyDev adds live status indicators to your terminal tabs when running AI CLI sessions like Claude Code or OpenAI Codex.

### What you get:

- **Terminal tab status** -- See at a glance which sessions are working and which need your input
- **Smart notifications** -- Get alerted when your AI agent has been waiting
- **Quick Reply** -- Respond without switching terminals

### How it looks:

```
  ⚡ Claude [a1b2] - Working      👀 Codex [c3d4] - Waiting
```
